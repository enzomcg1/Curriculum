Proyectos

Integración de Página Web con Base de Datos en PHPMyAdmin

Características Principales

Login y Registro de Usuarios:

Se implementó un sistema de autenticación que permitía a los usuarios registrados acceder a la plataforma mediante un nombre de usuario y contraseña. Los usuarios tenían la opción de registrarse proporcionando información básica como nombre, correo electrónico y contraseña. Esta información se almacenaba en la base de datos para su posterior acceso.

Index Principal:

El index principal de la página web servía como punto de entrada para los usuarios autenticados y visitantes. En esta sección, se exhibían los productos de la empresa de manera organizada y atractiva, utilizando CSS para mejorar la presentación visual. Cada producto se presentaba con información relevante, como nombre, descripción y precio, obtenida directamente de la base de datos.

Integración con Base de Datos en PHPMyAdmin:

Se utilizó PHPMyAdmin para gestionar la base de datos que almacenaba la información de usuarios y productos. Mediante consultas SQL, se estableció la conexión entre la página web y la base de datos, permitiendo la recuperación y manipulación de datos de manera eficiente.

Tecnologías Utilizadas

HTML, CSS y JavaScript para el desarrollo del frontend de la página web. PHP para la lógica del backend y la interacción con la base de datos. MySQL como sistema de gestión de base de datos, administrado a través de PHPMyAdmin.
